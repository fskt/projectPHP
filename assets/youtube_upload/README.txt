Berikut ini adalah tutorial singkat cara upload video ke youtube menggunakan Youtube API V3
1. buatlah project di google developer console (https://console.developers.google.com/project)
2. aktifkan module youtube data api
3. buatlah credential untuk akses aplikasi kita ke youtube
4. masukan alamat website uploader kita
5. buatlah server key agar kita otomatis login saat upload, tidak perlu ada popup login ke google
6. masukan credential yang kita dapat ke config aplikasi
7. upload source code ke server
8. sekarang kita coba jalankan di server
9. sebelumnya hanya ada 3 video
10. proses authentifikasinya masih salah, perbaiki redirectnya.. tapi sebenernya ini sudah login
11. kita ulangi lagi.. harusnya kali ini masuk
12. proses upload berhasil, harunya video ini sudah masuk ker server.
akan tetapi youtube perlu beberapa saat sampai selesai prose uploadnya dr server kita
